
## 题目

```
描述

给出一个字符串数组S，找到其中所有的乱序字符串(Anagram)。如果一个字符串是乱序字符串，那么他存在一个字母集合相同，但顺序不同的字符串也在S中。

所有的字符串都只包含小写字母
您在真实的面试中是否遇到过这个题？  
样例

对于字符串数组 ["lint","intl","inlt","code"]

返回 ["lint","inlt","intl"]
挑战

What is Anagram?

    Two strings are anagram if they can be the same after change the order of characters.


```
